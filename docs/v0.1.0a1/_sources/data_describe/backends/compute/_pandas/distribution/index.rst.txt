.. function:: compute_distribution(data, diagnostic=True, spike_factor=10, skew_factor=3, **kwargs)

   Compute distribution metrics.

   :param data: The data
   :type data: DataFrame
   :param diagnostic: If True, will compute diagnostics used to select "interesting" plots.
   :type diagnostic: bool, optional
   :param max_categories: Maximum categories to display. Defaults to 20.
   :type max_categories: int, optional
   :param label_name: The label to use for categories combined after max_categories. Defaults to "(OTHER)".
   :type label_name: str, optional
   :param spike_factor: The spikey-ness factor used to flag spikey histograms. Defaults to 10.
   :type spike_factor: int, optional
   :param skew_factor: The skew-ness factor used to flag skewed histograms. Defaults to 3.
   :type skew_factor: int, optional

   :returns: DistributionWidget
