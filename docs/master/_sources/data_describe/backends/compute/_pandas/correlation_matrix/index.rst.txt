.. function:: compute_correlation_matrix(data, cluster=False, categorical=False)

   Correlation matrix of numeric variables.

   :param data: The data frame
   :type data: DataFrame
   :param cluster: If True, use clustering to reorder similar columns together
   :type cluster: bool
   :param categorical: If True, calculate categorical associations using Cramer's V, Correlation Ratio, and
                       Point-biserial coefficient (aka Matthews correlation coefficient). All associations (including Pearson
                       correlation) are in the range [0, 1]
   :type categorical: bool

   :returns: CorrelationMatrixWidget

.. function:: cramers_v(x, y)

   Calculates Cramer's V statistic for categorical-categorical association.

   Adapted from https://github.com/shakedzy/dython/blob/master/dython/nominal.py

   :param x: A list, numpy array, or pandas series of categorical measurements
   :param y: A list, numpy array, or pandas series of categorical measurements

   :returns: Cramer's V value (float)
