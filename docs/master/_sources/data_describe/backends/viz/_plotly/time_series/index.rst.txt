.. function:: viz_plot_time_series(df, col, result=None, decompose=False, title='Time Series')

   Create timeseries visualization.

   :param df: The dataframe
   :param col: Column of interest. Column datatype must be numerical.
   :type col: str or [str]
   :param result: The statsmodels.tsa.seasonal.DecomposeResult object. Defaults to None.
   :param decompose: Set as True to decompose the timeseries with moving average. result must not be None. Defaults to False.
   :param title: Title of the plot. Defaults to "Time Series".

   :returns: The visualization
   :rtype: fig

   :param dates: The datetime index
   :param title: Title of the plot. Defaults to "Time Series Decomposition".

   :returns: The visualization
   :rtype: fig


.. function:: viz_plot_autocorrelation(data, white_noise, n_lags, plot_type='acf', title='Autocorrelation Plot')

   Create timeseries autocorrelation visualization.

   :param data: numpy.ndarray containing the correlations
   :param white_noise: Significance threshold
   :param plot_type: Choose between 'acf' or 'pacf. Defaults to "pacf".
   :param title: Title of the plot. Defaults to "Autocorrelation Plot".

   :returns: The visualization
   :rtype: fig
